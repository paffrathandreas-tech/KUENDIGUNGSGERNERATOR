# Kündigungsgenerator

Ein einfacher und kostenloser Web-Generator zur Erstellung von Kündigungsschreiben für Verträge wie Fitnessstudio, Handyvertrag oder Internetanbieter.

---

## 🚀 Funktionen

- ✅ Ordentliche Kündigung mit Frist
- ✅ Außerordentliche Kündigung mit Begründung
- ✅ Automatische Berechnung des Kündigungsdatums
- ✅ Manuelles Kündigungsdatum möglich
- ✅ Auswahl von Anbietern (z. B. Vodafone, Telekom, McFIT)
- ✅ PDF-Download des Kündigungsschreibens
- ✅ Versand per E-Mail möglich
- ✅ Modernes und übersichtliches Design

---

## 🛠️ Technologien

- React
- Vite
- Tailwind CSS
- jsPDF

---

## 📱 Verwendung

1. Persönliche Daten eingeben  
2. Vertragsdetails auswählen  
3. Kündigungsart bestimmen  
4. Kündigung erstellen  

Danach kann das Schreiben:
- kopiert  
- als PDF gespeichert  
- per E-Mail verschickt werden  

---

## 💡 Idee

Viele Kündigungs-Tools im Internet sind kompliziert, langsam oder kostenpflichtig.  
Dieses Projekt bietet eine einfache, schnelle und kostenlose Lösung.
